# 🚀 Anshu Kumar Portfolio

A modern, animated developer portfolio built using **Next.js 16**, **Tailwind CSS**, and **Framer Motion**. This project showcases my skills, projects, and experience as a Full Stack Developer.

---

## 🌐 Live Demo

👉 [https://anshu-kumar-portfolio-snowy.vercel.app/](https://anshu-kumar-portfolio-snowy.vercel.app/)

---

## 📌 Features

* ⚡ Modern UI with dark theme
* 🎬 Smooth animations using Framer Motion
* 📱 Fully responsive design (mobile + desktop)
* 🧭 Smooth section navigation
* 📂 Dynamic project pages using Next.js routing
* 💼 Clean developer-focused layout

---

## 🛠️ Tech Stack

* Next.js (App Router)
* React.js
* Tailwind CSS
* Framer Motion
* TypeScript

## ⚙️ Installation & Run Locally

```bash

```

```bash

npm start
```
